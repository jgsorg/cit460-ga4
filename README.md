# JavaScipt + GA4 Event Tracking Project

## Project Overview
This project demonstates the integration of JavaScript with Google Analytics 4 to track and analyze user interactions on a website. The goal is to gain hands-on experience with modern web analytics by implementing event-based tracking and verifying that user behavior data is successfully collected and displayed in the GA4 dashboard

This application consists of an interactive website that will track user events like page views, button clicks, etc. The project focuses on understanding how analytic data is generated and analyzed rather than building complex application features.

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Analytics 4
- GitHub Pages (deployment)

## Features
- JavaScript-based user interaction tracking
- GA4 integration
- Event tracking for user actions
- Public deployment for real-world data collection
